# Nensi Thummar - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring a dark theme with cyan and purple accents.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern Dark Theme**: Navy blue background with cyan/purple accent colors
- **Smooth Animations**: Fade-in effects, skill bar animations, and hover transitions
- **Interactive Elements**: Mobile hamburger menu, contact form, and button effects
- **Professional Layout**: Clean sections for About, Skills, Projects, and Contact

## Setup Instructions

1. **Download/Clone**: Get all files in the same directory structure
2. **Open**: Open `index.html` in any modern web browser
3. **Customize**: Replace placeholder content with your actual information
4. **Deploy**: Upload to GitHub Pages, Netlify, or any web hosting service

## Customization Guide

### Personal Information
- Replace "Nensi Thummar" with your name throughout the files
- Update the bio, tagline, and contact information
- Replace placeholder profile image with your photo

### Projects
- Update project cards with your actual projects
- Replace placeholder images with project screenshots
- Add real GitHub and demo links

### Colors (in style.css)
- Primary background: `#000021` (navy blue)
- Secondary background: `#000033` (darker blue)
- Accent color: `#00ffff` (cyan)
- Secondary accent: `#ff00ff` (purple)

### Skills
- Modify skill percentages in both HTML and JavaScript
- Add or remove skills as needed
- Update skill categories

## Deployment

### GitHub Pages
1. Create a new repository
2. Upload all files
3. Go to Settings > Pages
4. Select source branch
5. Your site will be available at `username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Your site will be live instantly with a custom URL
