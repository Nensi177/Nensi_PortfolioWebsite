# Portfolio Website

This is my personal portfolio website that I made while learning **HTML, CSS, and JavaScript**.  
It has a dark navy blue theme with cyan and purple colors. I created it to practice web design, animations, and responsive layouts.

## Features
- Works well on mobile, tablet, and desktop  
- Dark theme with smooth transitions  
- Simple and clean sections for About, Skills, Projects, and Contact  
- Some basic animations and hover effects  

## Sections

### Home/About
Includes my name, short introduction, and buttons for resume and hire me.  

### Skills
Shows my technical and soft skills with small animations.  

### Projects
Displays a few sample projects with short descriptions.  

### Contact
Simple contact form and social media links.  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Font Awesome  
- Google Fonts (Poppins)  

## What I Learned
- How to make a responsive website using Flexbox and media queries  
- How to add simple animations and effects  
- How to organize files and folders properly  
- How to upload a project on GitHub
